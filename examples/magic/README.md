# Magic

When working with Jupyter notebooks you can use "magic" commands to make your
work more efficient.  Note that these should *not* be used in production.


## What is it?

1. `magic.ipynb`: Jupyter notebook illustrating a number of magic commands.
1. `magic.py`: paired Python script.
1. `data.txt`: sample data file used in the notebook.
1. `gcd.py`: sample Python module used in the notebook.
