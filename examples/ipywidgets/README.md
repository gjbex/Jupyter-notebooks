# Widgets

Jupyter Lab supports a plethora of widgets and packages that can be used in
notebooks to facilitate interactive data exploration and visualization.


## What is it?

1. `widgets.ipynb`: an example Jupyter notebook that illustrates several
   widgets and packages, e.g., `ipywidgets`, `altair`, `matplotlib.animation`,
   `ipydatagrid`, etc.
1. `widgets.py`: paired Python sciprt.
1. `hanoi.ipynb`: an illustration of `matplotlib.animation` to visualize the
   Tower of Hanoi problem.
1. `hanoi.py`: paired Python script.
